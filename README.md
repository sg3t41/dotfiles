# Dotfiles

## Install

```bash
git clone https://github.com/sg3t41/dotfiles dotfiles
cd dotfiles
```

## SetUp

```bash
make local        # ローカルインストール
make docker       # Docker build + run
make docker-build # Docker build のみ
make docker-run   # Docker run (build 含む)
make clean        # Docker イメージ削除
make help         # ヘルプ表示
```
